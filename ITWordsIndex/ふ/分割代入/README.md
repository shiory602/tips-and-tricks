## 分割代入とは？
ストラクチャ (Structure) は「構造」・「構造物」という意味。

デストラクチャは、「『組み立て』の反対」つまり、「壊す」・「分解する」といった意味。

> 分割代入は、「配列の要素」や「オブジェクトのプロパティ」を分解・分割して、変数（や定数）に値を代入すること

- 配列は [ ] 記号
[配列の分割代入](../は/配列の分割代入)
- オブジェクトは { } 
[オブジェクトの分割代入](../お/オブジェクトの分割代入)
